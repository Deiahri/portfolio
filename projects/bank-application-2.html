Bank Application V2

Mock bank app - In Development

Goal

Follow the Agile Cowboy Methodology to develop a modern bank application, and demonstrate an understanding of Node.js, PostgreSQL, Database Development, and Agile Development.

Implementation

Used PostgreSQL to implement the bank's database, and Express.js (a Node.js framework) to communicate with the database. Implemented a front-end using BootStrap and JavaScript to make requests to the REST API, allowing the application to communicate with the database.
